# Introduction

SocketDB is an easy to use data storage that syncs accross browsers in realtime and allows you to focus on your application without worring about networking.

It features:

  • minimal network overhead: caches data and only syncs changes
  • batches changes to reduce transmitted data
  • automatically reconnects on connection lost
  • full typescript support
