# Quick Start

# Prequesites

SocketDB requires NodeJS.

# Installation

Note

Currently SocketDB is only distributed as a node module, so you can not simply link it in your html file. Instead it is recommended to use a bundler, like RollupJS (opens new window) for your frontend application.

For both client and server:

    yarn add socketdb

    npm install socketdb

    // Make sure to add code blocks to your code group

    # Setup Server

    Create index.js file with:

    import { SocketDBServer } from 'socketdb';

SocketDBServer({ port: 8080 });

    Run it with node index.js

    # Setup Client

    In your frontend javascript, you can connect to the server like this:

    import { SocketDBClient } from 'socketdb';

const db = SocketDBClient({ url: 'ws://localhost:8080' });

db.get('some')
	.get('path')
	.on(console.log);
db.get('some').set({ path: 'Hello World' });
